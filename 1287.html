Errata : Appendix A

 

The SQL table definition script for the Dat_Issue table is missing a

definition for the ModifiedDate column. The correct table definition should

be:

 

 

CREATE TABLE [dbo].[Dat_Issue] (

            [IssueID] [int] NOT NULL ,

            [TypeID] [int] NULL ,

            [UserID] [int] NULL ,

            [EntryDate] [datetime] NULL ,

            [StatusID] [int] NULL ,

            [Summary] [char] (128) COLLATE SQL_Latin1_General_CP1_CI_AS NULL ,

            [Description] [text] COLLATE SQL_Latin1_General_CP1_CI_AS NULL ,

            [PriorityID] [int] NULL ,

            [ModifiedDate] [datetime] NULL

) ON [PRIMARY] TEXTIMAGE_ON [PRIMARY]

GO
